# AGENTS.md

## Project goal

This repository contains my personal research portfolio website. We are building a redesigned v2 site while preserving the current published v1 site.

## Deployment safety

- Treat `main` as production unless the user says otherwise.
- Do not change GitHub Pages settings, DNS, custom domain settings, CNAME, or deployment workflows unless explicitly asked.
- Do not push directly to the production branch.
- Work on `redesign/research-portfolio-v2` or a PR branch.

## Content preservation

Preserve all existing research content:
- Publications
- Talks
- Projects
- CV links
- Bio
- Contact information
- Research descriptions
- Images and PDFs already used by the site

If content is reorganized, keep equivalent information discoverable.

## Design constraints

- Use inspiration sites only as references for layout, hierarchy, typography, spacing, and interaction feel.
- Do not copy proprietary code, images, logos, or highly distinctive visual assets.
- Keep the design professional, clean, fast, accessible, and appropriate for a theoretical physics research portfolio.
- Prefer readable typography, strong publication/project organization, and mobile-first layout.

## Build and test

Before calling work complete:
- Run the local build command.
- Fix broken links, missing assets, and console errors where practical.
- Verify homepage, publications, projects, CV, and contact routes.
- Check mobile layout.
